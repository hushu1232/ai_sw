AI 公司已经进入一个新的阶段， 过去大家拼的是模型能力
谁参数更强，谁上下文更长（一次可以给大模型的大小），谁跑分更高（计算，推理，编程...）
谁写代码更厉害，现在是谁真正把AI落地在企业与业务里。

企业买AI不是买API，也不是买token，企业到的是降本提效，增长，风控，
是一个能真正嵌入业务流程，每天被员工使用，最后能产生业务结果的系统。

FDE 更像是一个站在客户现场的一个AI工程负责人。

工作内容：
-理解客户的业务流程，越垂直越好
-哪些环节适合AI来改造
-把企业的内部的数据，系统工具连接起来
-搭建Agent，工作流，知识库，和评测体系
-确保这些东西在生产环境跑起来

从一千的卖模型，到卖结果。
